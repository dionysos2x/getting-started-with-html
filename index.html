The Basics of the Web

The world wide web (www) is basically a huge collection of HTML documents,
located on many different computers which communicate via HTTP (HyperText Transfer Protocol),
the main protocol of the web.
The many HTML files are connected via hyperlinks (URLs).
Web browsers can interpret the HTML files and display them for the user in a more appealing way.
The web was invented in the 1990s and today consists of about 30 Billion pages.

HTML

HTML stands for HyperText Markup Language.
It is made up of:
- text content (what you see)
- markup (describes how the text should look like)
- references to other documents (e.g. images, videos and music)
- links to other pages

Elements and Tags

HTML code is made of elements which are defined by tags.
There are many different HTML tags e.g. markup tags to make text bold or italic,
the anchor tag to create links like www.udacity.com
or the image tag to include images like this cute fellow

Other tags are for creating whitespace like linebreaks
or paragraphs.

Most HTML elements are defined by an opening tag and a closing tag,
but some elements don't need a closing tag. Those are called void tags.

Attributes

HTML Attributes are found inside tags and give the tag more variety and options.
The anchor tag's "href"-attribute for example is needed to define the URL for a link to another webpage.

Inline and Block Elements

HTML elements can be categorized as "inline" or "block", where block elements form an "invisible box"
around their content e.g. the paragraph tag.

The HTML Document Structure

In general the structure of an HTML document is as follows:
- Definition of the doctype (doctypes bring different presettings for the HTML document)
- Head (title and implementation of CSS and JavaScript can be defined here)
- Body (the main content of the webpage)
